## What git is?
It's *virtual code system* which means it stores history of the changes you make in your code so that if you want to go back to what you wrote before, like 2 days before, you can go to it and undo the changes you did in 2 days.
## How to push code to github?
Github is a website where you can use git and store your code and share your code with others. "push code to github" means using commandline commands or statements to add files and connect changes.
